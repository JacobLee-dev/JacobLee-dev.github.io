### Problem 11: Even or Odd
- **Link:** [Programmers 12937](https://school.programmers.co.kr/learn/courses/30/lessons/12937)

```python
def solution(n):
    if n % 2 == 0:
        return "Even"
    elif n % 2 != 0:
        return "Odd"
    
print(solution(3))
print(solution(4))
```

###Problem 13: Sum of Digits
- **Link:** [Programmers 12931](https://school.programmers.co.kr/learn/courses/30/lessons/12931)

```Python
def solution(N):
    total = 0
    while N > 0:
        total += N % 10
        N //= 10
    return total

print(solution(123))
print(solution(987))
```

### Problem 14: Sum of Divisors
- **Link:** [Programmers 12928](https://school.programmers.co.kr/learn/courses/30/lessons/12928)

```Python
def solution(n):
    total = 0
    for i in range(1, n + 1):
        if n % i == 0:
            total += i
    return total

print(solution(12))
print(solution(5))
```

### Problem 15: Find the Smallest Number with Remainder 1
- **Link:** [Programmers 87389](https://school.programmers.co.kr/learn/courses/30/lessons/87389)

```Python
def solution(n):
    for x in range(2, n):
        if n % x == 1:
            return x

print(solution(10))  # 3
print(solution(12))  # 11
```
