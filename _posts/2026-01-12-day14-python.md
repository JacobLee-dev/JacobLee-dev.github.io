### Problem 8: Protractor
- **Link:** [Programmers 120829](https://school.programmers.co.kr/learn/courses/30/lessons/120829)

```Python
def solution(angle):
    if 0 < angle < 90:
        return 1
    elif angle == 90:
        return 2
    elif 90 < angle < 180:
        return 3
    else:
        return 4
        
print(solution(70))
print(solution(91))
print(solution(180))
```

### Problem 9: Sum of Even Numbers
- **Link:** [Programmers 120831](https://school.programmers.co.kr/learn/courses/30/lessons/120831)

```Python
def solution(n):
    total = 0
    for i in range(2, n+1, 2):
        total += i
    return total

```
