## 💻 Today's SQL Solutions

### Problem 1: Find Animals with Names
- **Link:** [Programmers 59407](https://school.programmers.co.kr/learn/courses/30/lessons/59407)

```sql
SELECT animal_id
FROM animal_ins
WHERE name IS NOT NULL
ORDER BY animal_Id ASC;
```

### Problem 2: Sort by Animal ID Descending
- **Link:** [Programmers 59035](https://school.programmers.co.kr/learn/courses/30/lessons/59035)

```SQL
SELECT name, DATETIME
FROM ANIMAL_INS
ORDER BY animal_id DESC;
```

### Problem 3: Count Distinct Names
- **Link:** [Programmers 59408](https://school.programmers.co.kr/learn/courses/30/lessons/59408)

```SQL
SELECT COUNT(DISTINCT NAME) AS count
FROM ANIMAL_INS
WHERE NAME IS NOT NULL;
```



### Problem 6: Find Duplicate Names
- **Link:** [Programmers 59041](https://school.programmers.co.kr/learn/courses/30/lessons/59041)

```SQL
SELECT NAME, COUNT(*) AS COUNT
FROM ANIMAL_INS
WHERE NAME IS NOT NULL
GROUP BY NAME
HAVING COUNT(*) >= 2
ORDER BY NAME;
```

### Problem 7: Find Sick Animals
- **Link:** [Programmers 59036](https://school.programmers.co.kr/learn/courses/30/lessons/59036)

```SQL

FROM animal_ins
WHERE intake_condition = 'Sick';
```

### Problem 8:Find the First Animal
- **Link:** [Programmers 59405](https://school.programmers.co.kr/learn/courses/30/lessons/59405)

```SQL

FROM animal_ins
ORDER BY datetime ASC
LIMIT 1;
```

### Problem 10: Find Young Animals
- **Link:** [Programmers 59037](https://school.programmers.co.kr/learn/courses/30/lessons/59037)
