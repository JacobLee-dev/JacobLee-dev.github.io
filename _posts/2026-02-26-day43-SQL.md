## 💻 1. Python Lists Overview

Lists are used to store multiple items in a single variable. They are **ordered**, **changeable**, and **allow duplicate** values.

```python
# Create and access a List
thislist = ["apple", "banana", "cherry", "apple"]
print(thislist)       # ['apple', 'banana', 'cherry', 'apple']
print(len(thislist))  # 4
```

- Indexed: The first item has index [0].
- Dynamic: You can add, change, and remove items after creation.
- Versatile: Can contain mixed data types (String, Int, Boolean).

### 💻 2. Python Tuples Overview
##  Tuples are similar to lists but are ordered and unchangeable (Immutable). They also allow duplicate values.

# Create a Tuple
thistuple = ("apple", "banana", "cherry")

# Important: Single item tuple requires a comma
single_item = ("apple",)

- Immutable: Once created, you cannot change, add, or remove items.
- Safety: Used when you want to ensure the data in the collection remains constant throughout the program.

Feature,List ([]),Tuple (())
Order,Ordered,Ordered
Changeable,Yes (Mutable),No (Immutable)
Duplicates,Allows duplicates,Allows duplicates
Speed,Relatively Slower,Relatively Faster


### 🔑 Key Points  
- Use Lists when you need a collection that will change or grow during the program.

- Use Tuples for fixed data sets to provide a level of security and potentially better performance.

- Remember that both are indexed starting from [0], making data retrieval straightforward for both types.
