## 💻 Today's SQL Solutions

### Problem 11: Sort by Multiple Criteria
- **Link:** [Programmers 59404](https://school.programmers.co.kr/learn/courses/30/lessons/59404)

```SQL
SELECT 
    animal_id,
    name,
    datetime
from animal_ins
order by name asc, datetime desc;
```

### Problem 12: Find Dogs with Specific Name
- **Link:** [Programmers 59047](https://school.programmers.co.kr/learn/courses/30/lessons/59047)

``` SQL
SELECT ANIMAL_ID, NAME
FROM ANIMAL_INS
WHERE ANIMAL_TYPE = 'Dog'
    AND NAME LIKE '%EL%'
ORDER BY NAME;
```

### Problem 13: Count Users with No Age
- **Link:** [Programmers 131528](https://school.programmers.co.kr/learn/courses/30/lessons/131528)

```SQL
SELECT 
    count(*) as users  
from user_info
where Age is null;
```

### Problem 14: Find Maximum Price
- **Link:** [Programmers 131697](https://school.programmers.co.kr/learn/courses/30/lessons/131697)

```SQL
SELECT MAX(PRICE) AS MAX_PRICE
FROM PRODUCT;
```

### Problem 15: Handle NULL Values in Name
- **Link:** [Programmers 59410](https://school.programmers.co.kr/learn/courses/30/lessons/59410)
``` SQL
SELECT 
    ANIMAL_TYPE,
    IFNULL(NAME, 'No name') AS NAME,
    SEX_UPON_INTAKE
FROM ANIMAL_INS
ORDER BY ANIMAL_ID;
```


### 🔑 Key Points
- Practiced sorting data using multiple criteria with ORDER BY (e.g., ascending by name, descending by datetime).

- Utilized the LIKE operator with wildcard % for specific string matching, and MAX() for finding the highest value.

- Handled missing data effectively using IS NULL for filtering and IFNULL() to replace null values with specific text.
